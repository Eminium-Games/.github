## 🎮 About us
**Eminium** is a French-speaking Minecraft server bringing together enthusiasts around various game modes such as **PvP-Factions, Semi-RP, Survival, and Minigames**. Our goal is to offer a unique, high-performance gaming experience for our community.

On this GitHub organization, we host the source code for our internal and open-source projects:
* 🔌 Custom **Minecraft Plugins** (Spigot / Paper).
* 🤖 **Discord Bots** for community management.
* 🌐 **Web Infrastructure** (Website, store, API).

## 🛠️ Our Technologies
We primarily use the following technologies to run the Eminium ecosystem:
* **Minecraft Development:** Java, Spigot/Paper API, Gradle/Maven
* **Web Development:** HTML/CSS, JavaScript/TypeScript, PHP, React/Vue.js
* **Database & Infra:** MySQL, Redis, Docker, Linux (Debian/Ubuntu)
